# TDS

Developed with Unreal Engine 4 TDS

This project was created using c++ and blueprints. A standard shooter with a top view.
A system of weapons, buffs and debuffs was introduced, as well as an inventory system was created.
Artificial intelligence is written for standard opponents and a boss with tower-like functions. There is an interface.
The game contains a cooperative walkthrough. The server type is listen.
